Story

You have been cursed and poisoned by an evil wizard.... unlucky.
Help is on the way with an antidote but will not arrive till after sunset.
Gather as many resources as possible before sunset, when the curse will teleport you down into the depths of the earth.
You must then make it back to the surface and find rescue before the poison kills you.

Rules

1 stick + 1 coal = 1 torch. Torches burnout after 20 seconds.
Stone blocks that are not next to an air cannot be mined.
Raw meat heals half a slot on hunger bar. Cooked meat heals 1 slot on hunger bar.
Crafting Jack Lanterns, Beds, Fishing Rods and Glowstone disabled.
Nether Portals, Ravines and Mineshafts disabled.
On quest completion or death, save game is automatically deleted.
Once rescue is found right click to complete quest.
Brightness locked at 0.

Installation

1. Backup save games and minecrft.jar (minecraft.jar is located in the bin directory, save games located in the saves directory)
2. Install ModLoader using the instructions in the link above.
3. Unzip Journey_x.x.x.x.zip. (Using a program like 7-Zip)
3. Copy the resources folder to your .minecraft directory. (If asked to merge say yes)
4. Copy the contents of JourneyJar.zip into your minecraft.jar. (Using a program like 7-Zip)
5. Alternatively use a program such as MCPatcher to add ModLoader and JourneyJar.zip

Removal

1. Replace minecraft.jar in bin directory with the one you backed up.
2. If using MCPatcher simply deselect JourneyJar.zip and click patch.

Additional Info

1. Once installed in the Create World screen you need to press the "Game Mode" button until it displays "Game Mode: Journey" then click "Create World" button.
2. If the mod has installed correctly you should see a loading screen as shown in the above screenshot. Loading gamemode can take anywhere between 30 seconds to 5 minutes. If you experience any problems loading or load fail click the "Regenerate Map button.
